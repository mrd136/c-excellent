POS NETWORK PRINTER ONLINE

Connect local ESC/POS network printer to Pos retail for online server using Odoo Printer Connector

This module depends on POS NETWORK PRINTER

Create Printer Connector

Go to Point of Sale → Configuration → Printer Connector

  • Specify Connector name
    •

Add orders network printer to Connector

Go to Point of Sale → Configuration → Network printer. Edit the selected printer and add Connector

Create Odoo Printer Connector user

Go to Settings → User Create new user and attribute the access to Printer Connector only

Odoo Printer Connector configuration

Go to zip directory in pos_network_printer_online directory Extract OdooPrinterConnector to computer in your local network directly related to printer and open printer.conf

  • Specify host = Odoo host
  • Specify port = Odoo port
  • Specify dbname = Odoo database name
  • Specify user = Printer connector user
  • Specify password = Printer connector password
  • Specify token = token copy from Printer Connector in Odoo
    •

Execute connector.exe. Keep Ticket print mode to Network in Point Of Sale configuration. The receipt will printed to local printer
