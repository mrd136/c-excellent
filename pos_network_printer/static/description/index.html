ESC/POS Network printer configuration

This module work ONLY with ESC/POS Network Printer. You don't need POS Box

Add network printer

Go to Point of Sale → Configuration → Network printers

  • Specify Printer Name
  • Specify IP Address of network printer
  • Specify Port of network printer. Default : 9100
    •

Add network printer in Pos configuration

Go to Point of Sale → Configuration → Point of sale. Edit the Pos and add printer in Network printers section

Set ticket print mode to Network

Go to Point of Sale → Configuration → Point of sale. Edit the Pos and set the ticket print mode

Printer & print spooler status

You can see in the Pos interface the printer & print spooler status

Print receipt

Click on the printer icon to reconnect printer
Click on the spooler icon to print unprinted receipt one by one
